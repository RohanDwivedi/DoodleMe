### Added
- Initial project structure with rqt plugin scaffold
- Claude AI agent with tool-use pipeline (OpenSCAD, URDF, BOM, wiring, KiCad)
- Dark theme QSS stylesheet
- Chat panel with streaming token display
- 3D STL viewer panel (pyvistaqt)
- Bill of Materials panel with CSV export
- Wiring diagram panel (Mermaid via QWebEngineView)
- ROS2 URDF publisher with live RViz2 preview
- Gazebo Harmonic spawn integration
- Session auto-save and crash recovery
- Settings dialog with API key management
- Status bar with per-service health indicators
- Debian packaging for ROS2 Jazzy
